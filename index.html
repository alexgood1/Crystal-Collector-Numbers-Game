GAME RULES

You're given a Random Game Number (19 - 120) to start the game.

Click on any of the four crystals to collect points, trying to match the Random Game Number given at start. Each Crystal has a secret value (1 - 12), and values are hidden until clicked.

To Win: Collect an equal amount of points to match the Random Game Number.

If your total over the amount of the Random Game Number, You Lose!

Each game has a unique Random Game Number, and the secret values of each crystal are unique to each game.
